# MineSweeperTribute
Old window game Mine sweeper tribute
<img width="911" height="732" alt="Jeu base carrée" src="https://github.com/user-attachments/assets/4c9fbf4b-6726-49bb-be7b-2fa5e389ca29" />

* Offers the classic square cells minefield or an hexagon based minefield
* minefield size customizable from 20 x 20 to 60 x 100
* different difficulties (determine mines percentage)
* Zoomable minefield
<img width="504" height="331" alt="Nouveau jeu" src="https://github.com/user-attachments/assets/c9d97a56-c0d2-4bda-a3c1-6144d3479dcc" />

* English and french translations provided
